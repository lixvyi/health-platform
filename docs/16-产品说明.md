# 产品说明

## 1. 产品名称

健康大数据应用创新研发中心门户系统 V1.0

## 2. 运行环境

| 组件 | 版本 |
|------|------|
| JDK | 17+ |
| MySQL | 8.0 |
| Redis | 7.x |
| Node.js | 18+（前端构建） |
| Docker | 20+（可选） |

## 3. 安装部署

### 开发模式

```bash
# 数据库
cd docker && docker compose up -d mysql redis

# 后端
cd health-portal-backend
# IDEA 运行 HealthPortalApplication

# 前端
cd health-portal-frontend
npm install && npm run dev
```

### 生产模式（Docker）

```bash
cd health-portal-frontend && npm run build
cd ../docker && docker compose up -d --build
```

访问 http://localhost

## 4. 产品模块

门户首页、新闻中心、通知公告、应用中心、卫生政策、健康知识库、关于我们、内容管理后台、AI 健康问答、数据统计看板。
