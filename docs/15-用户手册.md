# 用户手册

## 一、公众门户（访客）

1. 浏览器访问 `http://localhost:5173`（或部署地址）  
2. **首页**：浏览轮播、新闻摘要、公告、应用入口  
3. **新闻中心/通知公告/卫生政策/健康知识库**：点击导航进入列表，点击标题查看详情  
4. **应用中心**：点击应用卡片跳转  
5. **关于我们**：查看中心介绍  
6. **AI问答**：输入健康科普问题，获取 AI 回复  

## 二、管理后台（编辑/管理员）

1. 访问 `http://localhost:5173/admin/login`  
2. 使用 `admin` / `Admin@123` 登录  
3. **数据看板**：查看各栏目内容统计图表  
4. **新闻/公告/政策/知识库**：新增 → 填写标题、内容 → 选择发布 → 保存  
5. **轮播管理**：维护首页轮播图  
6. **应用中心**：维护应用名称、图标、链接  
7. **关于我们**：编辑首页简介与关于页面内容  

## 三、常见问题

| 问题 | 处理 |
|------|------|
| 登录失败 | 确认后端已启动，账号 Admin@123 |
| 前台无数据 | 确认内容状态为「已发布」 |
| AI 无真实回答 | 配置环境变量 AI_API_KEY |
