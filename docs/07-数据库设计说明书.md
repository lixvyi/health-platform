# 数据库设计说明书

## 1. CDM 概念模型

**实体**：用户、内容、分类、轮播、应用、站点配置、AI对话记录  

**关系**：
- 用户 1:N 创建 内容  
- 分类 1:N 内容（通过 category_code 关联）  
- 用户 1:N AI对话记录  

## 2. PDM 物理表

| 表名 | 说明 |
|------|------|
| sys_user | 系统用户 |
| cms_category | 内容分类字典 |
| cms_content | 新闻/公告/政策/知识库统一内容 |
| cms_banner | 首页轮播 |
| cms_app | 应用中心 |
| cms_site_config | 关于我们、首页简介 |
| ai_chat_history | AI 问答历史 |

详见 `sql/init.sql`，可使用 PowerDesigner 逆向工程导入生成 ER 图。

## 3. 索引设计

- `cms_content.idx_cat_status_pub (category_code, status, publish_time)`：列表查询  
- `ai_chat_history.idx_session (session_id)`：会话追溯  

## 4. 初始化数据

- 管理员 admin、编辑 editor  
- 四类分类、示例内容、轮播、应用、关于我们
