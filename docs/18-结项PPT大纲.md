# 结项答辩 PPT 大纲

1. **封面**：项目名称、团队成员、指导教师  
2. **团队分工**（1 页）：A 后端 / B 前端 / C 测试  
3. **项目背景**（1 页）：健康大数据中心、门户定位  
4. **需求与用例**（1 页）：8 大功能模块用例图  
5. **架构设计**（1 页）：三端 + Docker 部署图  
6. **数据库设计**（1 页）：ER 图 / 核心表  
7. **功能演示**（3 页）：门户截图、后台截图、AI 问答  
8. **核心代码**（1 页）：JWT 过滤器 / ContentService  
9. **测试与性能**（1 页）：JUnit + JMeter  
10. **创新点**（1 页）：SpringAI + Redis + ECharts  
11. **总结与展望**（1 页）  

## 10 分钟演示脚本

1. 打开公众门户首页（30s）  
2. 浏览新闻详情（30s）  
3. 后台登录发布一条公告（2min）  
4. 前台刷新看到新公告（30s）  
5. AI 问答演示（1min）  
6. 数据看板（30s）  
7. 架构与分工总结（2min）
