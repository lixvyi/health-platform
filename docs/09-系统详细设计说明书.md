# 系统详细设计说明书

## 1. 登录时序图

```
用户 -> 前端: 提交用户名密码
前端 -> AuthController: POST /api/auth/login
AuthController -> AuthService: login()
AuthService -> SysUserMapper: findByUsername
AuthService -> PasswordEncoder: matches
AuthService -> JwtTokenProvider: generateToken
AuthController -> 前端: token + 用户信息
前端 -> localStorage: 存储 token
```

## 2. 发布内容时序图

```
编辑 -> AdminContentController: POST /api/admin/contents
AdminContentController -> ContentService: save()
ContentService -> CmsContentMapper: insert
ContentService -> Redis: 清除 stats 缓存
访客 -> PortalController: GET /api/portal/contents
PortalController -> ContentService: pagePublished()
```

## 3. 核心类

| 类 | 职责 |
|----|------|
| AuthService | 认证业务 |
| ContentService | 内容/轮播/应用/配置 |
| JwtTokenProvider | Token 生成解析 |
| AiChatService | SpringAI 调用与历史 |
| PortalController | 公众 API 聚合 |

## 4. AI 模块设计

- System Prompt 限定健康科普场景  
- 无 API Key 时演示模式返回提示
