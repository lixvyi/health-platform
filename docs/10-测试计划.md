# 测试计划

## 1. 测试范围

- 功能：8 大门户模块 + 后台管理 + 登录 + AI 问答  
- 性能：首页、内容列表 JMeter 压测  
- 单元：AuthService 等核心 Service  

## 2. 测试环境

| 项 | 配置 |
|----|------|
| OS | Windows 11 |
| JDK | 17 |
| MySQL | 8.0 Docker |
| 浏览器 | Chrome 最新 |

## 3. 通过标准

- 功能用例通过率 ≥ 95%  
- 无 P0/P1 未关闭缺陷  
- 50/100 并发压测无大量超时  

## 4. 进度

| 阶段 | 活动 |

| 构建 | 执行功能测试、单元测试 |
