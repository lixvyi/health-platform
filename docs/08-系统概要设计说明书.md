# 系统概要设计说明书

## 1. 模块划分

| 模块 | 包路径 | 职责 |
|------|--------|------|
| auth | module.auth | 登录、JWT |
| content | module.content | 内容、轮播、应用、站点配置 |
| portal | content.controller.PortalController | 公众只读 API |
| admin | content.controller.AdminContentController | 后台管理 API |
| file | module.file | 文件上传 |
| ai | module.ai | SpringAI 问答 |

## 2. 接口清单（核心）

| 方法 | 路径 | 说明 |
|------|------|------|
| POST | /api/auth/login | 登录 |
| GET | /api/portal/home | 首页聚合 |
| GET | /api/portal/contents | 内容分页 |
| GET | /api/portal/contents/{id} | 内容详情 |
| GET | /api/portal/apps | 应用列表 |
| GET | /api/portal/about | 关于我们 |
| POST | /api/ai/chat | AI 问答 |
| CRUD | /api/admin/contents | 内容管理 |
| CRUD | /api/admin/banners | 轮播管理 |
| CRUD | /api/admin/apps | 应用管理 |
| PUT | /api/admin/about | 关于我们配置 |
| GET | /api/admin/stats | 统计数据 |

## 3. 分层结构

Controller → Service → Mapper → Entity
